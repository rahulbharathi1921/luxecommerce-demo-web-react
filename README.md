## Tech Stack

| Layer | Technology |
|-------|------------|
| Framework | React 19 + TypeScript |
| Build Tool | Vite 6 |
| Styling | Tailwind CSS 4 |
| 3D / WebGL | Three.js + React Three Fiber + Drei |
| Animations | Framer Motion + React Spring |
| State | Zustand |
| Routing | React Router v6 |
| Icons | Lucide React |

## Run Locally

```bash
npm install
npm run dev
```

Runs at `http://localhost:3000`.

## Features

- Immersive 3D hero section with interactive torus knot & floating orbs
- Product browsing with search, sort, and 3D tilt hover cards
- Product detail pages with smooth layout animations
- Full checkout flow with cart management (Zustand)
- Lookbook masonry gallery with lightbox viewer
- Newsletter signup modal with localStorage persistence
- Custom cursor + mouse trail with hover text overlays
- Smooth page transitions & scroll reveal animations
- Fully responsive mobile-first design
